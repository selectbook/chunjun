# chunjun

chunjun(纯钧) 是一个基于Java, Soring Boot, Netty, Reactor, Hive, Spark, Flink等技术实现的企业级工业物联网基础平台，提供开箱即用
的物联网相关基础功能，低代码开发，设备集成，帮助企业快速创建行业物联网系统。
